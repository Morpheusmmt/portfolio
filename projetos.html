Site Universidade

No projeto UNES, desenvolvi uma aplicação web que inclui várias páginas, como uma seção de contato. Utilizei HTML para a estruturação, CSS para a estilização e JavaScript para implementar a funcionalidade de envio de e-mail ao preencher os campos de contato. Essa aplicação demonstra minhas habilidades em desenvolvimento web e interatividade, criando uma experiência mais dinâmica para os usuários.

Mega-Sena

Este projeto é uma aplicação web simples que simula o sorteio dos números da Mega-Sena, o popular jogo de loteria brasileiro. O usuário pode gerar números aleatórios para realizar apostas. A aplicação é construída com HTML e pode ser expandida com JavaScript para incluir recursos como histórico de números e análise de probabilidades. É uma ótima opção para quem está iniciando em desenvolvimento web e deseja praticar a geração de números aleatórios.

Previsão do tempo

Este projeto é uma aplicação web que permite consultar a previsão do tempo. Utilizando HTML, CSS e JavaScript, a aplicação consome dados de uma API externa para mostrar informações como temperatura, umidade e condições climáticas em tempo real. Uma ótima demonstração das minhas habilidades em integração com APIs e criação de interfaces dinâmicas e interativas.
